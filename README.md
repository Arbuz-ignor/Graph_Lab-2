`К лабораторной работе приложен тестовый файл "TESTgraph", с помощью которого можно проверить все функции структуры данных. Чтобы загрузить его в программу, нужно выбрать соответствующую цифру в CLI меню и затем написать точное название файла.`

Структура граф сделана на Java. Вся логика построена на собственных структурах данных - DynamicArray, HashMap, Queue, Stack; граф хранится в собственной хеш-таблице, обходы сделаны на стеке и очереди, чтение/запись реализованы вручную через GraphIO, а управление всем проектом идёт через консольный интерфейс GraphCLI.


Реализованы: собственные структуры данных, полный класс Graph с рёбрами, обходами(DFS и BFS) и Дейкстрой(поиск кратчайшего пути), выводы всех ребер, вывод смежных вершин, система ввода/вывода в файл, консольный интерфейс, обработка ошибок.
